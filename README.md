# Auto Text Effect Animation

A simple auto text animation built using **HTML, CSS, and JavaScript**.
The text updates automatically, showing different roles one character at a time, creating a typing effect.

## Live Demo

(https://hooriashujaat098.github.io/auto-text-effect-animation/)

## Features

* Auto typing text animation
* Smooth text update using JavaScript timing
* Multiple roles displayed in a loop
* Clean and minimal UI

## What I Learned

* How to select elements using `querySelector`

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

* Roles are stored in an array
* JavaScript updates the text character by character
* Once one role is completed, it moves to the next
* The animation loops continuously
